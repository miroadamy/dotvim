Installation:

    git clone git://github.com/radegast/dotvim.git

Create symlinks:

    ln -s ~/.vim/_vimrc ~/.vimrc
    ln -s ~/.vim/_gvimrc ~/.gvimrc

Switch to the `~/.vim` directory, and fetch submodules:

    cd ~/.vim
    git submodule init
    git submodule update
